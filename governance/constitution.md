# COGNITUM Constitution

1. Privacy-First: Zero-Retention ist unverrückbar.
2. Per-Sensor-Consent: Jeder Sensorkanal individuell steuerbar. Kamera/Mikrofon default OFF.
3. Local-First: Keine Cloud-Abhängigkeit.
4. Morphologisches Gate: Jede Funktionsentscheidung durchläuft VDI 2221/2222/2225.
5. ADR-Pflicht: Keine Implementierung ohne dokumentierte Architekturentscheidung.
6. Halal-Gate: AAOIFI 6-Faktor-Screening auf alle Revenue-Streams.
7. Zakat: 2.5% automatische Berechnung auf alle Einnahmen.
8. OTS-First: OpenTimestamps-Anker vor jeder Veröffentlichung.
9. SAFE Mode: Externes Backup vor jeder destruktiven Aktion.
10. Golden ZIP: Vollständige One-Shot-Deliverables.
